How cloud storage platform make money: 

Many files we save nowadays are not personal, they can be popular music or movies. So for a file, let's say it have a size of 1GB. Though seems like it took you 1GB of storage, indeed, for the cloud storage platform, they only use an ID to indicate that you have the same file. The ID is nothing but a number, it could be a few bytes. They just saved the file once, then use it everywhere. Eventually, you thought you bought a 2TB disk, but indeed, you have only used 10GB of that disk. The rest storage space is what the storage platform get from you.

假设一个硬盘2T，如果你只存独一无二的个人文件，网盘就赚不到钱了。但如果你保存一些公共文件，如流行歌曲或电影，网盘就只需要存一份原始文件，再给你的存储空间贴个标签，说你拥有对那个文件的访问权。表面上你占用了很大的空间，实际上你就占用了一个字符串。网盘就是这么挣钱的。这相当于你买了2TB的硬盘，实际上只用了10个GB。剩下的空间送给了网盘提供商。
